# Changelog

Все значительные изменения в проекте ShineCore Launcher будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-XX-XX

### Добавлено
- Полностью переработанный пользовательский интерфейс
- Поддержка кастомных фонов (изображения и видео)
- Система мониторинга процессов Minecraft
- Автоматическая загрузка и управление Java
- Консоль отладки с подробными логами
- Поддержка Fabric Loader
- Система модпаков с серверной синхронизацией
- Многоязычное приветствие
- Работа в системном трее

### Изменено
- Архитектура приложения на модульную систему
- Улучшена безопасность с Context Isolation
- Оптимизирована производительность загрузки
- Переработана система прогресса загрузки

### Исправлено
- Проблемы с запуском Fabric версий
- Ошибки распаковки нативных библиотек
- Проблемы с кодировкой на Windows
- Утечки памяти в интерфейсе

## [1.0.0] - 2025-XX-XX

### Добавлено
- Базовая функциональность лаунчера
- Поддержка ванильных версий Minecraft
- Простая система загрузки файлов
- Основной интерфейс пользователя
